# 화면 명세서 (Screen Specifications)

**프로젝트명**: 인플루언서 맞춤형 공동 여행 플랫폼
**문서 버전**: 1.0
**작성일**: 2025-01-18

---

## 문서 구성

이 문서는 3개 앱의 주요 화면에 대한 상세 명세를 제공합니다.

- Part 1: 유저 앱 화면 명세
- Part 2: 인플루언서 앱 화면 명세
- Part 3: 관리자 대시보드 화면 명세

---

# Part 1: 유저 앱 화면 명세

## 1. 인증 관련 화면

### US-001: 스플래시 화면 (SplashScreen)

**경로**: `/`

**화면 설명**: 앱 시작 시 로고와 로딩 표시

**UI 구성**:
- 서비스 로고 (중앙)
- 로딩 애니메이션
- 버전 정보 (하단)

**기능**:
- 토큰 유효성 확인
- 유효: 홈으로 이동
- 무효: 온보딩/로그인으로 이동

---

### US-002: 온보딩 화면 (OnboardingScreen)

**경로**: `/onboarding`

**화면 설명**: 서비스 소개 슬라이드 (최초 실행 시)

**UI 구성**:
- 이미지 슬라이드 (3-4장)
  - 슬라이드 1: "좋아하는 인플루언서와 함께"
  - 슬라이드 2: "안전한 결제와 정산"
  - 슬라이드 3: "특별한 여행 경험"
- 페이지 인디케이터
- [건너뛰기] 버튼 (우상단)
- [시작하기] 버튼 (마지막 슬라이드)

**기능**:
- 스와이프로 슬라이드 전환
- 로컬 스토리지에 온보딩 완료 저장

---

### US-003: 로그인 화면 (LoginScreen)

**경로**: `/login`

**화면 설명**: 소셜 로그인 옵션 제공

**UI 구성**:
- 서비스 로고
- 서비스 간단 소개 텍스트
- [카카오로 시작하기] 버튼 (노란색)
- [네이버로 시작하기] 버튼 (녹색) - 선택적
- 이용약관/개인정보처리방침 링크 (하단)

**기능**:
- 카카오 OAuth 인증 시작
- 로그인 성공 시 콜백 처리

**연관 API**:
- `POST /auth/kakao`

---

### US-004: 약관 동의 화면 (TermsAgreementScreen)

**경로**: `/terms-agreement`

**화면 설명**: 필수/선택 약관 동의

**UI 구성**:
- [전체 동의] 체크박스
- 구분선
- [필수] 서비스 이용약관 + [보기] 링크
- [필수] 개인정보 수집/이용 동의 + [보기] 링크
- [필수] 개인정보 제3자 제공 동의 + [보기] 링크
- [선택] 마케팅 정보 수신 동의
- [다음] 버튼 (필수 동의 시 활성화)

**기능**:
- 전체 동의 시 모든 항목 체크
- 개별 항목 체크 가능
- 약관 상세 보기 모달/화면

**연관 API**:
- `POST /users/terms-agreement`

---

### US-005: 본인인증 화면 (IdentityVerificationScreen)

**경로**: `/identity-verification`

**화면 설명**: 휴대폰 본인인증 진행

**UI 구성**:
- 안내 텍스트 "안전한 서비스 이용을 위해 본인인증이 필요합니다"
- 본인인증 필요 사유 설명
- [본인인증 하기] 버튼
- 인증 진행 중 로딩 표시

**기능**:
- 본인인증 모듈 연동 (PASS/KG이니시스)
- 인증 성공: 프로필 설정으로 이동
- 인증 실패: 오류 팝업 표시

**팝업**:
- 인증 실패 안내 팝업
- 인증 재시도 안내

**연관 API**:
- `POST /auth/identity-verification`

---

### US-006: 프로필 설정 화면 (ProfileSetupScreen)

**경로**: `/profile-setup`

**화면 설명**: 기본 프로필 정보 입력

**UI 구성**:
- 프로필 이미지 업로더 (원형, 카메라 아이콘)
- 닉네임 입력 필드 (중복 확인)
- 생년월일 선택 (날짜 피커)
- 성별 선택 (남/여/선택안함)
- [완료] 버튼

**기능**:
- 이미지 선택/크롭
- 닉네임 중복 확인
- 프로필 저장

**팝업**:
- 닉네임 중복 안내
- 설정 완료 안내

**연관 API**:
- `GET /users/check-nickname`
- `POST /users/profile`
- `POST /storage/upload`

## 2. 메인 화면

### US-007: 홈 화면 (HomeScreen)

**경로**: `/home`

**화면 설명**: 메인 랜딩 페이지

**UI 구성**:
- 헤더
  - 서비스 로고
  - 알림 아이콘 (뱃지)
- 서비스 소개 배너 (스와이프)
- "추천 코스" 섹션
  - 코스 카드 목록 (가로 스크롤)
- "진행 중인 파티" 섹션
  - 파티 카드 목록 (가로 스크롤)
- 하단 탭 네비게이션

**기능**:
- 배너 자동 스와이프
- 코스/파티 카드 클릭 시 상세 이동
- 알림 아이콘 클릭 시 알림 목록 이동

**연관 API**:
- `GET /banners`
- `GET /courses/featured`
- `GET /parties/active`

---

### US-008: 코스 상세 화면 (CourseDetailScreen)

**경로**: `/courses/:id`

**화면 설명**: 코스 전체 정보 표시

**UI 구성**:
- 상단 이미지 갤러리 (스와이프)
- 기본 정보 섹션
  - 코스 제목
  - 인플루언서 프로필 (사진, 이름, 클릭 가능)
  - 여행지 (국가, 도시)
  - 일정 (시작일 ~ 종료일, N박 M일)
  - 참가비
  - 모집 기간
  - 모집 현황 (00/00명)
- 탭 네비게이션
  - [일정] 탭: 일자별 상세 일정
  - [포함/불포함] 탭: 포함/선택 사항
  - [숙소] 탭: 숙소 정보
- 하단 고정
  - [문의하기] 버튼
  - [신청하기] 버튼 (Primary)

**일정 탭 상세**:
- Day 1, Day 2... 아코디언
- 각 항목: 시간, 제목, 설명, [지도보기] 버튼

**포함/불포함 탭**:
- 포함 사항 리스트 (체크 아이콘)
- 불포함 사항 리스트 (엑스 아이콘)
- 선택 사항 리스트 (안내 아이콘)

**숙소 탭**:
- 숙소 이미지
- 숙소명, 설명
- [지도에서 보기] 버튼

**기능**:
- 이미지 갤러리 확대 보기
- 구글맵 링크 외부 이동
- 신청 조건 확인 (나이/성별)
- 모집 상태 확인

**팝업**:
- 참가 조건 안내 팝업
- 환불 정책 안내 팝업
- 모집 마감 안내 팝업
- 이미 신청함 안내 팝업

**연관 API**:
- `GET /courses/:id`
- `GET /courses/:id/days`

---

### US-009: 코스 신청 화면 (CourseApplyScreen)

**경로**: `/courses/:id/apply`

**화면 설명**: 코스 신청 폼 및 결제

**UI 구성**:
- 코스 요약 정보 (상단)
  - 코스명, 일정, 참가비
- 신청 폼
  - 이름 (자동 입력, 수정 가능)
  - 연락처 (자동 입력, 수정 가능)
  - 인스타그램 주소 (필수) *
  - 나이 (자동 계산/수정 가능)
  - 성별 (자동/수정 가능)
  - 자기소개/신청 동기 (텍스트 영역)
- 약관 동의
  - 결제 및 환불 정책 동의
- [결제하기] 버튼

**기능**:
- 인스타그램 URL 유효성 검증
- 참가 조건 확인 (미충족 시 안내)
- 결제 모듈 연동 (네이버페이/카드)

**팝업**:
- 참가 조건 미충족 안내
- 결제 확인 팝업
- 결제 취소 확인 팝업

**연관 API**:
- `POST /courses/:id/apply`
- `POST /payments/request`

- `GET /my/applications?type=course`
